# Returns Module

This module handles all return shipment functionality.

## Structure

- **models/** - Data models for returns
- **providers/** - State management providers for returns
- **screens/** - UI screens for returns management
- **widgets/** - Reusable widgets specific to returns

## Features

- View all return shipments
- Return details and tracking
- Return status updates
- Return processing and confirmation

## Usage

Import the returns screen:
```dart
import 'package:now_delivery/features/business/shipments/returns/screens/returns_screen.dart';
```
