# Orders Module

This module handles all delivery order-related functionality.

## Structure

- **models/** - Data models for orders
- **providers/** - State management providers for orders
- **screens/** - UI screens for orders management
- **widgets/** - Reusable widgets specific to orders

## Features

- View all delivery orders
- Order details and tracking
- Order status updates
- Delivery confirmation

## Usage

Import the orders screen:
```dart
import 'package:now_delivery/features/business/shipments/orders/screens/orders_screen.dart';
```
