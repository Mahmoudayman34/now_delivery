# Pickups Module

This module handles all pickup request functionality.

## Structure

- **models/** - Data models for pickups
- **providers/** - State management providers for pickups
- **screens/** - UI screens for pickups management
- **widgets/** - Reusable widgets specific to pickups

## Features

- View all pickup requests
- Pickup details and scheduling
- Pickup status updates
- Collection confirmation

## Usage

Import the pickups screen:
```dart
import 'package:now_delivery/features/business/shipments/pickups/screens/pickups_screen.dart';
```
